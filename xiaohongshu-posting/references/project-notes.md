# Project Notes

## Scope

- Input: multiple desk/study photos from user.
- Output: 1-3 title/body/tag sets, then iterate based on user feedback.
- Optional: web publishing flow after explicit confirmation.

## Style decisions

- Theme: late-night study desk, warm light, walnut desk, Mac/book/stationery/life items.
- Tone: real, lived-in, not overly "chicken soup" or mechanical.
- Must include: concrete method details + short emotional reflection.
- Emojis: required but restrained.
- Body length: about one phone screen with blank lines between paragraphs.

## Tags

- Default: #书桌日常 #学习日常 #桌面好物 #自我提升
- Adjust only when the content clearly calls for it.

## Titles

- No fixed structure; rotate skeletons for variety:
  - Method + result
  - Scene + takeaway
  - Counter-intuitive + small breakthrough

## Publishing

- Web flow only; confirm before any action.
- Initial skill includes a checklist-style guide; automation details TBD.
